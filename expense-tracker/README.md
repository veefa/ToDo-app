# Expense Tracker

A simple Expense Tracker built with **React**, **TypeScript**, and **Tailwind CSS**.

## Features

- Add and view expenses
- Automatically clear form after submitting
- Styled using Tailwind CSS

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Vite (for fast development)
